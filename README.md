Welcome to my cybersecurity journey — a structured, hands-on roadmap where I’m building real-world skills in **networking, security analysis, penetration testing, and scripting**.

To become a **Threat Intelligence & Security Researcher** with strong practical knowledge in:

* Network security
* Offensive security (penetration testing)
* Security analysis & investigation
* Automation using Python & Bash

📁 **Hands-on Labs**

* Practice from platforms like TryHackMe

📁 **Scripts & Tools**

* Custom Python tools (port scanner, automation scripts, etc.)

📁 **Mini Projects**

* Integrated tools and workflows
