# Topic - Linux file system and using commands(2)
Objective : Getting familiar with advanced commands

Concept :
Linux uses a hierarchical file system and most Linux systems work better on Command Line Interface(CLI). These commands can be used to list files , navigate through directories, read and edit the file and managing users and system information.

Commands : 
ps - process status
ps aux - list all running processes with details
top -u <username> - monitor specific user
chmod - change mode
chown - change owner
nano  - edit files
wget - download files from web via http
scp - transfers file over ssh between two devices securely
kill - terminate specific process
systemctl - interacts with system

User and group info :
users contains : userID , root user , system user 
groups contains : Primary , Secondary
file permissions : read (4), write(2), execute(1)
