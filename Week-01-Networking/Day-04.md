# Topic - linux file system aand using commands(1)
Objective : Getting familiar with basic commands

Concept :
Linux uses a hierarchical file system and most of the linux system works better on Command Line Interface(CLI). These commands can be used to list the file , navigate through directories, read and edit the file and managing users and system info.

Commands :
pwd - current location
ls - list
ls -l - detailed list
ls -al - hidden files list
cd - change directory
cat - read a file 
find - search files and directories
whoami - current user name
uname -a - details of os
df -h - disk space
grep - searches text inside files
wc - word count
& - run commands in background
&& - used to chain commands in one line, executes next only if previous succeeds
(>) - redirect output to a file
touch - create file
su  - switch user
/var - variable data directory
/etc - root directory
/tmp - temporary file directory
/root - root user
